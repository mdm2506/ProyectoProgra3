Botones

¿Para qué sirven? Permiten al usuario seleccionar la opcion dependiendo del caso,ya sea para ingresar a otro apartado o realizar una accion en especifico.

Código HTML:


<button class="primario">Primario</button>
<button class="secundario">Secundario</button>
<button class="outline">Outline</button>
<button class="disabled">Disabled</button>

Inputs & Fields Section

¿Para qué sirven? En esta parte el cliente podra ingresar datos como un texto breve,correo,contraseña,y si lo necesitara, un area de texto para emitir un comunicado más largo.

Código HTML:


<div class="form-interior">
    <label for="input-text">texto</label>
    <input type="text" id="input-text" class="form_input" required placeholder="">
</div>

<div class="form-interior">
    <label for="email">email</label>
    <input type="email" id="email" class="form_input" placeholder="ejemplo.ejemplo@gmail.com" required>
</div>

<div class="form-interior">
    <label for="password">password</label>
    <input type="password" id="password" class="form_input" placeholder="password" required>
    <div>
        <textarea name="" id="" maxlength="100" rows="8"></textarea>
    </div>
</div>

Formularios section

¿Para qué sirven? Los formularios permiten al usuario ingresar información, seleccionar opciones o enviar datos que serán procesados. Se utilizan en casos como registro de usuarios, contacto, pedidos o cualquier acción que requiera interacción directa entre el usuario y el sistema.

¡Formulario!

mensaje

Por favor ingresa un email válido
Debes seleccionar una opción
0 / 500
El campo no puede estar vacío

Código HTML:


<form class="formulario-contenedor">
  <h4 class="formulario-titulo">¡Formulario!</h4>
  <p class="formulario-subtitulo">mensaje</p>

  <!-- INPUT TEXTO VACÍO -->
  <div class="campo-form">
    <label for="ejemplo-vacio" class="etiqueta-form">Campo vacío *</label>
    <input type="text" id="ejemplo-vacio" class="input-form" placeholder="Escribe aquí...">
  </div>

  <!-- INPUT TEXTO CORRECTO -->
  <div class="campo-form campo-form-correcto">
    <label for="ejemplo-correcto" class="etiqueta-form">Campo correcto *</label>
    <input type="text" id="ejemplo-correcto" class="input-form" value="Juan Carlos Pérez">
  </div>

  <!-- INPUT TEXTO ERROR -->
  <div class="campo-form campo-form-error">
    <label for="ejemplo-error" class="etiqueta-form">Campo con error *</label>
    <input type="email" id="ejemplo-error" class="input-form" value="email-invalido">
    <div class="mensaje-error-form">Por favor ingresa un email válido</div>
  </div>

  <!-- SELECT ERROR -->
  <div class="campo-form campo-form-error">
    <label for="opciones-error" class="etiqueta-form">Opción con error *</label>
    <select id="opciones-error" class="select-form">
      <option value="">Selecciona una opción</option>
      <option value="1">Opción A</option>
      <option value="2">Opción B</option>
    </select>
    <div class="mensaje-error-form">Debes seleccionar una opción</div>
  </div>

  <!-- TEXTAREA ERROR -->
  <div class="campo-form campo-form-error">
    <div class="etiqueta-y-contador">
      <label for="mensaje" class="etiqueta-form">Escribe tu mensaje aquí *</label>
      <div class="contador-caracteres-form">0 / 500</div>
    </div>
    <textarea id="mensaje" name="message" class="textarea-campo" placeholder="Escribe tu mensaje..." required maxlength="500"></textarea>
    <div class="mensaje-error-form">El campo no puede estar vacío</div>
  </div>

  <!-- BOTONES -->
  <div class="acciones-form">
    <button type="button" class="disable-button" disabled>Enviar (deshabilitado)</button>
    <button type="button" class="primary-button">Enviar (activo)</button>
    <button type="button" class="disable-button" disabled>Enviado</button>
  </div>
</form>

Labels section

¿Para qué sirve? Las etiquetas (labels) son elementos de texto que proporcionan información descriptiva sobre otros elementos de la interfaz de usuario, como campos de formulario, botones o secciones específicas. Ayudan a los usuarios a comprender el propósito y la función de los elementos con los que interactúan, mejorando la usabilidad y accesibilidad del sitio web.

Código HTML:


<label for=""  class="label Grande">Label Grande</label>
<label for="username" class="label">Label Normal</label>
<label for="" class="label Pequeño">Label Pequeño</label>

Titulo Principal 3,1em


<h1 class="TituloPrincipal">Titulo Principal</h1>

Hola soy un h2 2.7em


<h2 class="TituloSecundario">Hola soy un h2 </h2>

Hola soy un subtitulo 2em


<h3 class="subtitulo">Hola soy un subtitulo</h3>

Links section

¿Para qué sirve? Los enlaces (links) son elementos interactivos que permiten a los usuarios navegar entre diferentes páginas web o secciones dentro de una misma página. Son fundamentales para la estructura de la web, facilitando el acceso a información adicional, recursos relacionados y funcionalidades específicas.

Este es un tipo número uno del link tipo 1

Este es un tipo número dos del link tipo 2

Código HTML:


<a href="#" class="tipo1">link tipo 1</a>
<a href="#" class="tipo2">link tipo 2</a>

Radio Buttons section

¿Para qué sirve? Los radio buttons permiten al usuario seleccionar una única opción de un grupo de alternativas mutuamente excluyentes. Solo se puede elegir una opción a la vez. Son ideales para configuraciones donde necesitas una selección obligatoria entre varias opciones.

Selecciona tu color favorito:

Código HTML:


<section id="RadioButtons">

  <h2>Esta es la sección de RadioButtons</h2>

  <p>Selecciona tu color favorito:</p>

  <label>
    <input type="radio" id="color-f08d7d"grise="color" value="f08d7grishecked>
    <span id="color-f08dgrisabel">Grisal</span>
  </label>

  <label>
    <input type="radio" id="color-f0b498" name="color" value="f0b498">
    <span id="color-f0b498-label">Melocotón</span>
  </label>

  <label>
    <input type="radio" name="color" value="rojo" disabled>
    <span id="color-rojo-label">Rojo (deshabilitado)</span>
  </label>

</section>

Check Boxes section

¿Para qué sirve? Los checkbox o casillas de verificación sirven para que un usuario pueda seleccionar una o varias opciones de un conjunto en un formulario o interfaz.


          <label for="check">Esto es un checkbox</label>
          <input type="checkbox" id="check">

Select/Dropdown section

¿Para qué sirve? Los elementos select/dropdown permiten a los usuarios elegir una opción de una lista predefinida de valores. Son ideales cuando hay múltiples opciones pero el espacio es limitado, ya que muestran solo la opción seleccionada hasta que el usuario hace clic.

Código HTML:


<section id="Select-Dropdown">
  <h2>Esta es la sección de Select/Dropdown</h2>

  <label for="opciones">Selecciona una opción:</label>
  <select class="opciones">
    <option value="1">Opción 1</option>
    <option value="2">Opción 2</option>
    <option value="3">Opción 3</option>
  </select>

  <select class="opciones" disabled>
    <option>Opción deshabilitada</option>
  </select>
</section>
